# iOS Security Review

Perform comprehensive security audit for iOS code.

## Check for:
1. **Hardcoded Secrets**
   - API keys in code
   - Passwords or tokens
   - Encryption keys

2. **Network Security**
   - HTTP vs HTTPS usage
   - Certificate validation
   - Man-in-the-middle vulnerabilities

3. **Data Storage**
   - Keychain usage for sensitive data
   - UserDefaults misuse
   - File encryption

4. **Authentication**
   - Biometric authentication
   - Password policies
   - Session management

5. **Code Injection**
   - Dynamic code execution
   - Web view security
   - Deep linking validation

6. **Privacy**
   - Permission usage
   - Data collection
   - Third-party SDKs

Generate a security report with findings and recommendations.
